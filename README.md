# iOS SDK

## Overview

`ringcaptcha/ringcaptcha-ios` is an iOS SDK for working with RingCaptcha API and RingCaptcha user interface components, which streamline and simplify the Onboarding and Verification of Users

## Getting Started

CocoaPods

```pod
pod 'Ringcaptcha'
```
